### Types

Type di golang sendiri terdiri dari

| Type       | Nilai          | Initial                                              |
| ---------- | -------------- | ---------------------------------------------------- |
| Booleans   | true dan false | bool                                                 |
| Floats     | 0.0            | float64, float32                                     |
| complex128 | 1.0i           | complex128                                           |
| Integers   | 1,2,3          | int, uint, int8, uint8, int16, uint16, int32, uint32 |
| chars      | 'a'            | char                                                 |
| string     | "string"       | string                                               |
